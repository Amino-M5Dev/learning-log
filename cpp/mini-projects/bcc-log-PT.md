# Aprendido (PT)
- Organização de código em múltiplas funções.
- Uso de passagem por referência (&).
- Separação entre entrada, processamento e saída.
- Uso de switch para seleção de operações.
- Validação básica (divisão por zero).
- Uso de fmod() para resto com números decimais.
- Estrutura de repetição com controle por variável (loop de continuidade).

## Feito (PT)
- Desenvolvimento de uma calculadora básica modular.
- Implementação de funções separadas para cada operação.
- Implementação de menu inicial.
- Controle de repetição do programa.
- Tratamento de erro para divisão e módulo por zero.
