# ROADMAP FRONT-END 2026 – FOCO EM MONETIZAÇÃO (FEV → JUN)

# OBJETIVO:
Construir base sólida em HTML, CSS e JavaScript até junho.
Estar apto a criar sites profissionais e começar a monetizar entre junho e setembro.

# FASE 1 – CONSOLIDAÇÃO DE HTML (1-2 semanas)

## FOCO:
- Estrutura semântica (header, nav, section, article, footer)
- Hierarquia correta de títulos
- Formulários completos (input types, validação básica)
- SEO básico (meta tags, alt em imagens)
- Acessibilidade básica

## O QUE VOCÊ DEVE SABER FAZER SEM CONSULTAR:
- Criar página estruturada do zero
- Criar formulário funcional
- Organizar conteúdo de forma semântica

## PROJETOS:
1. Página pessoal simples (bio + contatos)
2. Landing page de produto fictício
3. Página institucional de empresa fictícia
4. Formulário de cadastro validado no HTML

# FASE 2 – CSS MODERNO E RESPONSIVO (3-4 semanas)

## FOCO:
- Box model profundo
- Flexbox (dominar completamente)
- CSS Grid
- Responsividade (media queries)
- Mobile first
- Transições e animações simples
- Organização de código CSS
- Boas práticas de layout

# O QUE VOCÊ DEVE SABER FAZER:
- Criar layout responsivo sem tutorial
- Centralizar qualquer elemento sem dor
- Construir layout completo com Grid ou Flexbox

## PROJETOS:
1. Recriar homepage de site conhecido
2. Landing page totalmente responsiva
3. Portfólio pessoal simples
4. Dashboard estático com layout moderno
5. Site institucional para pequeno negócio fictício

# FASE 3 – JAVASCRIPT FUNDAMENTOS (4-5 semanas)

## FOCO:
- Variáveis (let, const)
- Tipos de dados
- Condições e loops
- Funções
- Arrays e objetos
- Métodos importantes (map, filter, reduce)
- Manipulação do DOM
- Eventos
- Validação de formulário via JS

## O QUE VOCÊ DEVE SABER FAZER:
- Resolver problemas lógicos simples
- Manipular elementos da página
- Criar interações dinâmicas

## PROJETOS:
1. To-do list funcional
2. Calculadora
3. Validador de formulário com mensagens de erro
4. Galeria de imagens interativa
5. Sistema simples de filtro de produtos

# FASE 4 – JAVASCRIPT INTERMEDIÁRIO (4 semanas)

## FOCO:
- Fetch API
- Promises
- Async/Await
- Consumo de API pública
- Manipulação avançada de arrays
- Modularização de código
- Armazenamento local (localStorage)

## O QUE VOCÊ DEVE SABER FAZER:
- Consumir API e exibir dados
- Criar aplicações que salvam estado
- Organizar código em múltiplos arquivos

## PROJETOS:
1. App de clima consumindo API
2. Lista de tarefas com persistência no localStorage
3. Sistema de busca de usuários via API
4. Catálogo de produtos com filtro e ordenação
5. Mini dashboard dinâmico

# FASE 5 – PROJETO PROFISSIONAL (MAIO → JUNHO)

## OBJETIVO:
- Construir algo que você realmente mostraria para cliente.

## FOCO:
- Código limpo
- Performance básica
- SEO básico
- Estrutura profissional
- Deploy (Netlify, Vercel ou similar)
- Domínio básico

## PROJETOS:
1. Site real para negócio fictício completo
2. Portfólio profissional definitivo
3. Sistema simples de agendamento
4. Landing page otimizada para conversão
5. Projeto final combinando:
   - HTML semântico
   - CSS moderno
   - JavaScript dinâmico
   - API externa

# REGRAS ESTRATÉGICAS

1. Não estudar só teoria.
2. Cada conceito novo → aplicar em mini projeto.
3. Não pular fundamentos para framework cedo demais.
4. Não buscar perfeição antes de praticar.
5. Foco em constância semanal.

# META ATÉ JUNHO

## Você deve conseguir:
- Criar site institucional completo sozinho
- Criar landing page responsiva sem copiar
- Usar JavaScript para interatividade real
- Consumir API
- Fazer deploy
- Ter 3-5 projetos apresentáveis
