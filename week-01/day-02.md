# Aprendido
- If / Else (estruturas condicionais).
- Switch (seleção múltipla).
- While loop (repetição com condição).
- For loop (repetição controlada).
- Nested loops (loops dentro de loops).
- Functions (declaração, definição e retorno de valores).

## Feito
- Exercício simples de verificação de número (positivo, negativo ou zero).
- Exercício de menu usando switch.
- Exercício de contagem com while.
- Exercício de repetição controlada com for.
- Exercício de impressão de padrões usando nested loops.
- Exercício de criação e uso de funções com retorno.
